StatsD Client
=============

A very simple StatsD client for PHP. Records timing, increment, and count metrics.

Install
-------

Require `bigcommerce/statsd-client` as a dependency in your `composer.json` manifest:

```
require: {
	"bigcommerce/statsd-client": "1.0"
}
```
